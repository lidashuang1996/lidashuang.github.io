1. 什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2. 安装 Hexo

依赖安装：

  1. Git 是一个开源的分布式版本控制系统,用于敏捷高效地处理任何或小或大的项目。

  2. Node.js 为大多数平台提供了官方的 安装程序。对于中国大陆地区用户，可以前往 淘宝 Node.js 镜像 下载， Hexo 官方文档。
