火猫-房间信息（ 弹幕信息、礼物信息、排行榜信息，房间贵族人数信息、等 ）是通过Web Socket 推送，我们可以通过 JAVA （ Web Socket Client ）来获取数据，然后制作出相应的插件，来扩展火猫服务。

1. 环境准备

  • 谷歌浏览器开发版
  • JAVA 开发环境 （采用 JAVA 开发）

2. 火猫房间协议

  • 消息长度：4 字节小端整数，表示整条消息（包括自身）长度（字节数）。

  • 消息头部总长度 ：2 字节。（固定为 16，例如 [ 0, 16 ]）

  • 消息协议 ：2 字节。

    • 0 服务器推送过来的数据
    • 1 提交给服务器的数据

  • 消息类型：4 字节小端整数，表示消息类型。取值如下：

    • 7 登录提交消息类型。
    • 8 登录返回消息类型。
    • 2 心跳提交消息类型。
    • 3 心跳返回消息类型。
    • 5 服务器推送消息类型。

  • 消息备用字段：4 字节小端整数，内容同消息协议。

  • 数据部分：发送的消息内容（消息内容按照 JSON 格式数据传输）
